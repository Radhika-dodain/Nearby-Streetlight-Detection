The "Safety Spotlight" Sensor-
The Goal: You will tell the Chef (FastAPI) your current location, and he will check the Pantry (PostGIS) to see if you are standing under a "Street Light." If you are within 500 meters of a light, the app says "You are in the light!" If not, it says "It's dark here, be careful."

Basic project to learn the workflow of backend and databases- used neon console, sql commands, postGIS and python FastAPI for backend.
